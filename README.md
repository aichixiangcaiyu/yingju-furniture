## How to Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders in this package to the repository root.
3. Go to repository **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. GitHub will generate a website URL like:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```

## Important Before Official Use

1. Replace AI-generated images with real factory, product, loading and project photos.
2. Replace certificate placeholders with real verified certificates or company documents.
3. Edit `sitemap.xml` and `robots.txt`:
   - Replace `YOUR-GITHUB-USERNAME`
   - Replace `YOUR-REPOSITORY-NAME`
4. Connect the contact form to email, WhatsApp, CRM or a real backend.
5. Replace placeholder product details with real dimensions, materials, MOQ and loading quantities.
